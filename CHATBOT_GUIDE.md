## How to Use

### Step 1: Open the Assistant

**Look for the floating button:**
- Located at **bottom-right** corner of screen
- Blue button with a **message icon** (ğŸ’¬)
- Bounces slightly to attract attention when closed

**Click to open:**
- Button turns red when opened
- Chat window slides up from bottom

### Step 2: Ask Your Question

**Examples of questions:**
```
1. "Explain the concept of Federalism in Indian Constitution"
2. "What is monetary policy and how does it affect inflation?"
3. "Describe the greenhouse effect and its role in climate change"
4. "How does the Indian electoral commission work?"
5. "What is the difference between budget deficit and trade deficit?"
```

**Type your question:**
- Use the input box at the bottom
- Type naturally (grammar doesn't have to be perfect)
- Press **Enter** or click **Send button**

### Step 3: Read the Response

The chatbot provides:
1. **Clear Definition** - What the concept is
2. **Key Points** - 2-3 most important aspects
3. **Real-World Example** - Concrete scenario
4. **UPSC Connection** - Why it matters for the exam
5. **Further Insights** - Related concepts or follow-ups

### Step 4: Ask Follow-Up Questions

- **Continue the conversation** - Context is maintained
- **Ask for clarification** - "Can you explain that differently?"
- **Request examples** - "Give me more examples"
- **Deep dive** - "What about the exceptions?"

## Common Questions

### Q: Is my chat history saved?
**A:** Chat history is maintained only during your current session. Once you close the browser, chat is cleared. This is by design for privacy.

### Q: Can I use this during exam?
**A:** No, this is only for study preparation. The actual exam requires your own knowledge.

### Q: What if the bot gives wrong answer?
**A:** Always verify important information from official sources (NCERT, PIB). The bot is a learning aid, not a replacement for research.

### Q: How long can my questions be?
**A:** You can ask multi-part questions, but shorter, focused questions usually get better responses.

### Q: Can the bot remember between sessions?
**A:** No. Each session starts fresh. If you want persistent notes, download and save using the download feature.

### Q: Is there a daily limit?
**A:** No limits on conversations. Chat as much as you need!

### Q: Can I chat offline?
**A:** No, chatbot requires internet connection to reach Gemini API.

### Q: Does chatbot track my questions?
**A:** No personal tracking. Questions are processed for AI response only and not stored.

## Troubleshooting

### Chatbot Not Responding

**Issue**: Clicking send but no response appears
- **Solution 1**: Check internet connection
- **Solution 2**: Verify API key is configured in .env.local
- **Solution 3**: Refresh page and try again
- **Solution 4**: Clear browser cache

### Error Message Appears

**Issue**: "API Key not configured"
- **Solution**: Ensure `GEMINI_API_KEY` is in your `.env.local` file

**Issue**: "Request timeout"
- **Solution**: Wait a moment and try again; server may be slow

### Chat Looks Weird

**Issue**: Messages overlap or format looks broken
- **Solution**: 
  - Refresh the page
  - Try a different browser
  - Clear browser cache

### Chatbot Gives Off-Topic Responses

**Issue**: Response doesn't relate to UPSC
- **Solution**: Rephrase question more specifically towards UPSC context
